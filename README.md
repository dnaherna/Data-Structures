# Data-Structures  

## Available  
---  
- Stack  
- Linked List  
- Binary Search Tree  
- Heap  

<br>

### Guide  
---  

To build all data structures run `make`  
To build a specific data structure run `make <data structure>`  

Alternatively, each directory has a Makefile
